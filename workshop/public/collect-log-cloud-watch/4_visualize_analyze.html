4. Visualize and Analyze

OpenSearch provides us the ability to analyze out logs. Lets begin by naviaging to the OpenSearch Dashboard

Open the OpenSearch Dashboard

  1. Go to the OpenSearch Console
  2. Click on the workshop-domain OpenSearch domain you created earlier

  1. Click on the OpenSearch Dashboard URL. This should open the URL in a web browser window

  1. You will be prompted to log in. Using the user name and password you created during the OpenSearch deployment, log in
  2. If an additonal pop up window is present after login asking about data upload click on Explore on my own
  3. If an additonal pop up windows is present asking you to select your tenant select Global and click on Confirm

You should now see a window that looks like this

Create an Index Pattern

The Lambda functions the send the messages from CloudWatch to OpenSearch will create a new OpenSearch index each day. Each index name will start with cwl and will be followed by the date.

To search all of the CloudWatch logs (ie. multiple days) we will create an index pattern in OpenSearch. The index pattern will be a representation of all of the cwl log indexes for all of the days.

  1. In the OpenSeach Dashboard, expand the side menu and click on Stack Management under managment section

  1. On the stack managment page click on Index Patterns on the left hand menu

  1. On the index patterns page click on Create index pattern

  1. Enter workshop-log-* under the index pattern name section
  2. Click on Next step

  1. Select @timestamp as the primary time feild
  2. Click on Create index pattern

We have now created an index pattern! We can use the index pattern to analyze our logs
